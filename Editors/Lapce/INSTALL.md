# Installing Spacecraft Software Theme for Lapce Editor

**Version:** 1.0 | **Author:** Mohamed Hammad | **License:** GPL3+ | **Website:** [SpacecraftSoftware.org](https://SpacecraftSoftware.org)

## Installation

1. Copy `spacecraft-software.toml` to your Lapce themes directory:
   - **Linux:** `~/.config/lapce-stable/themes/`
   - **macOS:** `~/Library/Application Support/dev.lapce.Lapce-Stable/themes/`
   - **Windows:** `%APPDATA%\dev.lapce.Lapce-Stable\themes\`
2. Open Lapce → **Command Palette** (`Ctrl+Shift+P`) → **Change Color Theme**.
3. Select **Spacecraft Software** from the list.
