# Installing Spacecraft Software Theme for Zed Editor

**Version:** 1.0 | **Author:** Mohamed Hammad | **License:** GPL3+ | **Website:** [SpacecraftSoftware.org](https://SpacecraftSoftware.org)

## Installation

1. Copy `spacecraft-software.json` to your Zed themes directory:
   - **macOS/Linux:** `~/.config/zed/themes/`
   - **Windows:** `%APPDATA%\Zed\themes\`
2. Open Zed → **Command Palette** (`Ctrl+Shift+P`) → **theme selector: Toggle**.
3. Select **Spacecraft Software** from the list.

## Alternative: Settings File

Add to your Zed `settings.json`:
```json
{
    "theme": "Spacecraft Software"
}
```
