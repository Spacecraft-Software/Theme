# Installing Spacecraft Software Theme for XFCE Terminal

**Version:** 1.0 | **Author:** Mohamed Hammad | **License:** GPL3+ | **Website:** [SpacecraftSoftware.org](https://SpacecraftSoftware.org)

## Installation

1. Copy `Spacecraft-Software.theme` to:
   - `~/.local/share/xfce4/terminal/colorschemes/`
   - Or system-wide: `/usr/share/xfce4/terminal/colorschemes/`
2. Open XFCE Terminal → **Edit** → **Preferences** → **Colors**.
3. Under **Presets**, select **Spacecraft Software**.
