# Installing Spacecraft Software Theme for Alacritty

**Version:** 1.0 | **Author:** Mohamed Hammad | **License:** GPL3+ | **Website:** [SpacecraftSoftware.org](https://SpacecraftSoftware.org)

## Installation

1. Copy `spacecraft-software.toml` to your Alacritty config directory:
   - **Linux:** `~/.config/alacritty/themes/`
   - **macOS:** `~/.config/alacritty/themes/`
   - **Windows:** `%APPDATA%\alacritty\themes\`
2. In your main `alacritty.toml`, add:
   ```toml
   import = ["~/.config/alacritty/themes/spacecraft-software.toml"]
   ```
3. Restart Alacritty.
