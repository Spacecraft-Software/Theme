# Installing Spacecraft Software Theme for GNOME Terminal

**Version:** 1.0 | **Author:** Mohamed Hammad | **License:** GPL3+ | **Website:** [SpacecraftSoftware.org](https://SpacecraftSoftware.org)

## Installation

1. Make the script executable:
   ```bash
   chmod +x spacecraft-software-gnome-terminal.sh
   ```
2. Run the script:
   ```bash
   ./spacecraft-software-gnome-terminal.sh
   ```
3. Open GNOME Terminal → **Preferences** → **Profiles** → select **Spacecraft Software**.
