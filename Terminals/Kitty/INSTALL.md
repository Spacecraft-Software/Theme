# Installing Spacecraft Software Theme for Kitty Terminal

**Version:** 1.0 | **Author:** Mohamed Hammad | **License:** GPL3+ | **Website:** [SpacecraftSoftware.org](https://SpacecraftSoftware.org)

## Installation

1. Copy `spacecraft-software.conf` to your Kitty config directory:
   - `~/.config/kitty/themes/`
2. In your `~/.config/kitty/kitty.conf`, add:
   ```
   include themes/spacecraft-software.conf
   ```
3. Restart Kitty (or press `Ctrl+Shift+F5`).
