# Installing Spacecraft Software Theme for iTerm2

**Version:** 1.0 | **Author:** Mohamed Hammad | **License:** GPL3+ | **Website:** [SpacecraftSoftware.org](https://SpacecraftSoftware.org)

## Installation

1. Double-click `Spacecraft-Software.itermcolors` — iTerm2 will import it automatically.
2. *Or* go to **iTerm2** → **Preferences** → **Profiles** → **Colors** → **Color Presets…** → **Import…** → select `Spacecraft-Software.itermcolors`.
3. Select **Spacecraft Software** from the Color Presets dropdown.
