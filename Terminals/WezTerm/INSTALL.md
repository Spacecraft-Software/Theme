# Installing Spacecraft Software Theme for WezTerm

**Version:** 1.0 | **Author:** Mohamed Hammad | **License:** GPL3+ | **Website:** [SpacecraftSoftware.org](https://SpacecraftSoftware.org)

## Installation

1. Copy `spacecraft-software.lua` to your WezTerm colors directory:
   - `~/.config/wezterm/colors/`
2. In your `~/.config/wezterm/wezterm.lua`, add:
   ```lua
   config.color_scheme = "spacecraft-software"
   ```
3. Save and WezTerm will reload automatically.
