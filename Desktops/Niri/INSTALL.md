# Installing Spacecraft Software Theme for Niri

**Version:** 1.0 | **Author:** Mohamed Hammad | **License:** GPL3+ | **Website:** [SpacecraftSoftware.org](https://SpacecraftSoftware.org)

## Installation

1. Merge the contents of `config.kdl` into your Niri config:
   - `~/.config/niri/config.kdl`
2. Copy `Spacecraft_Software_wallpaper_blue.png` to your wallpapers directory.
3. Update the wallpaper path in `config.kdl` if necessary.
4. Reload Niri config.
