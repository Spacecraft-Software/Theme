# Installing Spacecraft Software Theme for MATE Desktop

**Version:** 1.0 | **Author:** Mohamed Hammad | **License:** GPL3+ | **Website:** [SpacecraftSoftware.org](https://SpacecraftSoftware.org)

## Installation

### Window Borders (Metacity)
1. Create the theme directories:
   ```bash
   mkdir -p ~/.themes/Spacecraft-Software/metacity-1/
   ```
2. Copy `metacity-theme-1.xml` to:
   - `~/.themes/Spacecraft-Software/metacity-1/`
3. Open **System** → **Preferences** → **Appearance** → **Customize** → **Window Border** → select **Spacecraft Software**.

### Wallpaper
1. Copy `Spacecraft_Software_wallpaper_blue.png` to `~/Pictures/`.
2. Right-click desktop → **Change Desktop Background** → select the wallpaper.
