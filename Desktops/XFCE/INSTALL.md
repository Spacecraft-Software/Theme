# Installing Spacecraft Software Theme for XFCE

**Version:** 1.0 | **Author:** Mohamed Hammad | **License:** GPL3+ | **Website:** [SpacecraftSoftware.org](https://SpacecraftSoftware.org)

## Installation

### GTK Theme
1. Create the theme directory:
   ```bash
   mkdir -p ~/.themes/Spacecraft-Software/gtk-3.0/
   ```
2. Copy `gtk.css` to:
   - `~/.themes/Spacecraft-Software/gtk-3.0/gtk.css`
3. Open **Settings** → **Appearance** → select **Spacecraft Software**.

### Wallpaper
1. Copy `Spacecraft_Software_wallpaper_blue.png` to `~/Pictures/`.
2. Right-click desktop → **Desktop Settings** → select the wallpaper.

### XFCE Terminal
See `Theme/Terminals/XFCE_Terminal/INSTALL.md` for terminal color scheme.
