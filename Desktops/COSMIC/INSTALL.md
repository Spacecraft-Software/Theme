# Installing Spacecraft Software Theme for COSMIC Desktop

**Version:** 1.0 | **Author:** Mohamed Hammad | **License:** GPL3+ | **Website:** [SpacecraftSoftware.org](https://SpacecraftSoftware.org)

## Installation

1. Copy `spacecraft-software.ron` to:
   - `~/.config/cosmic/com.system76.CosmicTheme.Dark/v1/`
2. Copy `Spacecraft_Software_wallpaper_blue.png` to your wallpapers directory.
3. Open **Settings** → **Desktop** → **Appearance** → select the Spacecraft Software theme.
4. Set wallpaper via **Settings** → **Desktop** → **Wallpaper**.
