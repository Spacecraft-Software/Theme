# Installing Spacecraft Software Theme for MangoWC

**Version:** 1.0 | **Author:** Mohamed Hammad | **License:** GPL3+ | **Website:** [SpacecraftSoftware.org](https://SpacecraftSoftware.org)

## Installation

1. Copy `theme.toml` to your MangoWC config directory:
   - `~/.config/mangowc/themes/spacecraft-software.toml`
2. In your MangoWC config, set:
   ```toml
   theme = "spacecraft-software"
   ```
3. Copy `Spacecraft_Software_wallpaper_blue.png` to your wallpapers directory.
4. Restart MangoWC.
